# Simple Todo List

A minimal Todo List app that allows you to add, edit, delete, and search todo items.

Live Demo: [https://advanced-todo-list-puce.vercel.app/](https://advanced-todo-list-puce.vercel.app/)

## Features

* Add new todos
* Edit existing todos
* Delete todos
* Search todos

## Getting Started

```bash
npm install
npm run dev
```
